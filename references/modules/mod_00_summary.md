# 模块 0：投资摘要（Executive Summary）

## 要求清单
- [ ] 投资评级（买入 / 条件性买入 / 持有 / 卖出）
- [ ] 当前价 / 目标价（悲观/基准/乐观）/ 止损价
- [ ] 核心投资逻辑（3条，每条≤40字，直击要害）
- [ ] 核心风险（2条，每条≤30字）
- [ ] 关键数据速览表（8-10项）

## 生成原则
- **必须最后写**：摘要是全报告的浓缩，须在模块1-10生成完毕后提炼，不可提前凑数
- **不可以是对正文的复制粘贴**：用自己的语言提炼，追求"30秒读完即知是否值得买"
- **目标价须与 5.1 节一致**；投资评级须与 8.1 节一致
- **核心逻辑须可证伪**：每条逻辑隐含一个可观测的反驳条件（即第9节失效清单的映射）
